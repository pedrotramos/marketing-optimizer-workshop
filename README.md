# Apps Immersive Day Workshop

## Objetivo

O objetivo desse workshop é proporcionar uma atividade hands-on em que os participantes possam explorar na prática as funcionalidades do Databricks Apps e suas integrações com diversos outros recursos Databricks.

## Pré-requisitos

Todos os participantes do workshop devem cumprir alguns requisitos. Todos devem ter:

- Acesso a um workspace com Unity Catalog habilitado.
- Permissão para criar um catálogo, um schema e tabelas.
- Permissão para utilizar uma SQL Warehouse.
- Permissão para a criação de Jobs.
- Permissão para a criação de um App.
- Permissão para a criação de uma base de dados Lakebase.

## Instruções

Acesse o diretório de *instructions* para seguir o passo a passo de configuração do app. Além disso, lá você encontrará instruções para completar os desafios propostos no workshop.
